# AlaHexEditor
Powerful Hex Editor

# Prerequisites
- Java 19
- Maven

# How to build
```
mvn install
```

# How to run
```
mvn javafx:run@run
```

# How to debug
You have to launch normally the following command (don't launch it in debug mode).
```
mvn javafx:run@debug
```
Then click "Attach debugger".

# How to run as jar
```
mvn install
java -jar .\target\AlaHexEditor-1.0-SNAPSHOT-jar-with-dependencies.jar
```
